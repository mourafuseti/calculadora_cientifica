# Calculadora Científica

![Calculadora Científica](https://raw.githubusercontent.com/mourafuseti/calculadora-em-python/main/tela_cientifica.png)  
*Interface moderna, tema escuro e funções avançadas*

Uma **calculadora científica completa** desenvolvida em **Python** com a biblioteca **`customtkinter`**. Possui design responsivo, botões grandes, hover effects e suporte a todas as operações científicas essenciais.

---

## Funcionalidades

| Categoria | Operações |
|---------|---------|
| **Básicas** | `+`, `−`, `×`, `÷`, `%`, `±` |
| **Trigonométricas** | `sin`, `cos`, `tan`, `sin⁻¹`, `cos⁻¹`, `tan⁻¹` |
| **Logarítmicas** | `log₁₀`, `ln` |
| **Potenciação** | `x²`, `√x`, `x^y` (`^`) |
| **Fatorial** | `x!` |
| **Constantes** | `π`, `e` |
| **Parênteses** | `( )` |
| **Modo Angular** | `DEG` / `RAD` (alternável) |
| **Outros** | `C` (limpar), `⌫` (apagar último), ponto decimal |

> Resultados com até **10 casas decimais** (arredondamento inteligente).

## Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/mourafuseti/calculadora_cientifica.git
cd calculadora_cientifica

# 2. Instale a dependência
pip install customtkinter

# 3. Execute
python calculadora_cientifica.py
```

> *Ou baixe apenas o arquivo `calculadora_cientifica.py` e execute diretamente.*

---

## Como Usar

1. **Digite números e operadores** com os botões.
2. Use **parênteses** para agrupar expressões.
3. **Funções científicas**: clique no botão (ex: `sin`, `log`) → digite o valor → `=`.
4. **Modo angular**: clique em `DEG` para alternar entre **graus** e **radianos**.
5. **Limpar**: `C` (tudo) ou `⌫` (último caractere).
6. **Erro**: exibe `"Erro"` em caso de operação inválida.

## Exemplos de Cálculo
